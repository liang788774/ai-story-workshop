# AI故事工坊 - 使用指南

## 🚀 快速开始

### 方法一：使用 VS Code + Live Server（推荐）

1. **安装 VS Code**
   - 访问 [VS Code 官网](https://code.visualstudio.com/) 下载并安装
   - 打开 VS Code

2. **安装 Live Server 插件**
   - 点击左侧的 **扩展** 图标（快捷键：`Ctrl+Shift+X`）
   - 在搜索框中输入 `Live Server`
   - 找到官方插件（作者：Ritwick Dey）并点击 **安装**

3. **打开项目文件夹**
   - 在 VS Code 中点击 **文件** > **打开文件夹**
   - 选择 `ai-story-workshop` 文件夹

4. **启动 Live Server**
   - 在文件资源管理器中找到 `index.html`
   - 右键点击 `index.html`
   - 选择 **Open with Live Server**（用 Live Server 打开）
   - 浏览器会自动打开项目首页

### 方法二：直接用浏览器打开

1. 找到 `ai-story-workshop` 文件夹
2. 双击 `index.html` 文件
3. 浏览器会打开项目（注意：某些功能可能受限）

## 🎮 功能介绍

### 1. 首页 (index.html)
- 查看项目介绍和功能特点
- 快速进入创作或浏览故事

### 2. 剧情编辑器 (editor.html)
- **添加节点**：点击 "+ 添加剧情节点" 创建新的剧情点
- **编辑节点**：双击节点进行编辑
- **拖拽节点**：自由移动节点位置
- **AI辅助**：在右侧面板输入想法，点击 "AI生成剧情"
- **导出故事**：完成后导出为 JSON 文件

### 3. 游戏游玩 (play.html)
- 自动加载示例故事
- 点击选项推进剧情
- 支持重新开始
- 自动存档功能

### 4. 故事库 (stories.html)
- 浏览精选故事
- 开始游玩故事

### 5. 个人中心 (profile.html)
- 查看创作记录
- 管理个人作品

## 🔧 开发说明

### 本地开发
- 项目使用纯前端技术（HTML, CSS, JavaScript）
- 本地开发时 AI 功能使用模拟数据
- 无需配置服务器环境

### 部署上线
查看详细的部署指南：[DEPLOY_GUIDE.md](./DEPLOY_GUIDE.md)

**快速部署步骤：**
1. 将项目上传到 GitHub 仓库
2. 在 Netlify 中导入 GitHub 仓库
3. 设置 API_KEY 环境变量
4. 一键部署，获得全球访问链接

## 🐛 常见问题

### Q: AI 功能不工作？
A: 本地开发时使用模拟数据，部署到 Netlify 后才能使用真实 AI 功能

### Q: 如何连接节点？
A: 当前版本支持节点创建和编辑，完整的连线功能正在开发中

### Q: 存档功能在哪里？
A: 游戏会自动保存进度到浏览器本地存储
